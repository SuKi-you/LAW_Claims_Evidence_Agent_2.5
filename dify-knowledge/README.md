# Dify 知识库导入文件

## 本目录用途

本目录由 Excel 种子表转换生成，用于手动导入 Dify 知识库。内容仅基于原 Excel 中已有字段进行清洗、分组和格式转换，未新增法条、司法解释、案例或外部规则。

## 文件说明

- `01_离婚基础规则.md`：离婚、感情破裂、婚姻关系基础、离婚登记、调解无效等基础规则。
- `02_子女抚养规则.md`：子女抚养权、抚养费、探望权、未成年人利益、孩子生活照顾等规则。
- `03_家暴与人身安全保护规则.md`：家暴、人身安全保护、保护令、威胁、恐吓、伤情、报警、离婚损害赔偿等规则。
- `04_财产分割规则.md`：夫妻共同财产、房产、车辆、存款、债务、财产分割等规则。
- `05_第三者赠与与财产转移规则.md`：第三者赠与、小三转账、异常转账、财产转移、银行流水等规则。
- `06_证据规则总表.md`：所有涉及证据目的、证明任务、证据类型或准备提示的条目汇总。
- `knowledge-index.jsonl`：一行一个 JSON 对象，便于后续程序化校验、索引或导入。

## 建议导入 Dify 的顺序

建议优先导入：

1. `01_离婚基础规则.md`
2. `02_子女抚养规则.md`
3. `03_家暴与人身安全保护规则.md`
4. `04_财产分割规则.md`
5. `05_第三者赠与与财产转移规则.md`
6. `06_证据规则总表.md`

## 建议 chunk 设置

- 使用中文语义检索友好的 chunk 方式。
- 每个 chunk 尽量保持一条规则完整。
- 如果可设置 chunk 大小，建议不要把多条规则混在一个 chunk 里。
- 如果 Dify 支持 Parent-child / hierarchical chunking，可以优先尝试。
- 不要为了减少文件数量把所有内容压成一个超长文本。
