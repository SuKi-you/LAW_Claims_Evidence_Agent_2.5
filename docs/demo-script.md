# 演示脚本（约 3 分钟）

## 准备

- 确保 `npm run dev` 已启动
- 打开浏览器访问 `http://localhost:5173`
- 打开 DevTools → Network 面板（可选，用于观察请求）

---

## 场景 1：单纯离婚（30 秒）

**输入：**
```
我想离婚，没有孩子，也没有共同财产
```

**观察点：**
- 三步进度条进入"诉求识别" → 出现候选诉求卡片
- 诉求卡片应只包含：**离婚**（明确诉求）
- 不应出现：子女抚养权、抚养费、财产分割、损害赔偿（因为用户明确表示无子女无财产）

**操作：**
- 勾选"离婚" → 点击"确认诉求并生成证据清单"

**验证：**
- 证据清单应只包含：身份证件、结婚证、户口簿、感情破裂相关材料
- 不应出现：银行流水、转账记录、第三者身份线索、伤情照片

---

## 场景 2：压力用例 — 仅确认离婚但原文含小三转账和威胁（45 秒）

**输入：**
```
我想离婚，他还偷偷给小三转钱，也经常威胁我
```

**观察点：**
- 诉求卡片可能出现多个候选（离婚 + 追回第三者赠与 + 家暴/人身安全保护 + 财产转移）
- 但用户**只勾选"离婚"**

**操作：**
- 只勾选"离婚" → 点击确认

**验证：**
- 证据清单**只显示离婚基础证据**
- **不得**出现：微信转账截图、银行流水、第三者身份线索、人身安全保护令申请材料、伤情照片、报警记录
- 这是 confirmed_claims 锁定证据范围的**核心压力用例**

---

## 场景 3：追回第三者赠与（45 秒）

**输入：**
```
我发现我老公偷偷给小三转了很多钱，我想追回
```

**观察点：**
- 诉求卡片应包含：追回第三者赠与

**操作：**
- 勾选"追回第三者赠与" → 点击确认

**验证：**
- 核心证据应包含：转账记录、银行流水、微信/支付宝支付凭证
- 辅助证据应包含：第三者身份线索
- 风险提示应包含：需证明转账系赠与第三者且违背公序良俗
- **不得**出现：子女抚养权材料、人身安全保护令材料、损害赔偿材料

---

## 场景 4：财产转移（45 秒）

**输入：**
```
离婚前我发现对方把夫妻共同财产偷偷转走了
```

**观察点：**
- 诉求卡片应包含：财产转移

**操作：**
- 勾选"财产转移" → 点击确认

**验证：**
- 核心证据应包含：银行账户流水、异常转账记录、夫妻共同财产凭证
- 辅助证据应包含：财产转移线索
- **不得**出现：子女出生证明、抚养费证明材料

---

## 总结要点（演示者口播）

1. **诉求不脑补** — 系统根据用户描述的事实生成候选诉求，不会凭空捏造
2. **用户控制权** — 用户手动勾选确认诉求，不确认的不生成证据
3. **证据不越界** — 即使用户原文提到了其他事实，只要没确认对应诉求，就不会出现在证据清单中
4. **安全第一** — API Key 全程不出现在浏览器中，通过服务端代理转发
